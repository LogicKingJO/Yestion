# Yestion(예스션)


## 프로젝트 소개:
"매번 계획만 세우고 끝났다면? 이제 예스션(Yestion)할 시간!"
예스션은 복잡한 할 일 관리에서 벗어나, 당신이 오늘 바로 행동할 수 있도록 돕는 스마트한 투두리스트 서비스입니다. 미루기만 했던 일들에 'YES'를 외치고, 체크 하나로 채워지는 일상의 성취감을 경험해보세요.

## 핵심 기능:
- 간편한 체크, 확실한 성취: 터치 한 번으로 할 일을 완료하고 시각적인 피드백을 받아요.
- 심플한 UI/UX: 복잡한 설정 없이, 오늘 해야 할 일에만 온전히 집중할 수 있는 화면을 제공합니다.
- 꾸준한 동기부여: 내가 실천한 기록들을 보며 매일 성장하는 나를 발견할 수 있어요.
